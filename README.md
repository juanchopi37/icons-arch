# Una serie de iconos para mejorar la experiencia de GNOME

![iconos](./assets/icons.png)

## Aplicaciones

- Zen Browser
- Zed IDE
- Inkscape
- Alacritty
- Local Send
- Notion (Cohesion)
- Blender
- Carpetas para nautilus (Developer y OneDrive)
